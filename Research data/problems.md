# Core Problems (Prioritized)

## P0 — Core Blockers

### 1. Endorsements are complex and error-prone

- Multiple templates required per action
- Small errors require full re-upload
- Heavy reliance on Excel + brokers
- No clear success/failure feedback

### 2. No real-time financial visibility

- CD balance not reliable or updated
- No premium impact before action
- Users depend on finance/Acko teams

### 3. Portal is not trusted as source of truth

- Users re-confirm data via calls
- Duplicate workflows (portal + Excel)
- Inconsistent system outputs

### 4. Workflows depend on manual processes

- HR → Excel → Broker → Portal
- Batch processing (2-week cycles)
- Manual validation before submission

### 5. No validation before submission

- Errors detected only after upload
- False success states
- No inline correction
